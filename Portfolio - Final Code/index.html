Web App, Mobile App, Business Concept

I worked in the aquatics industry for years and realized that most (if not all) swim schools are unable to scale quickly because they depend on expensive, brick-and-mortar facilities. The solution for this is to have a team of mobile instructors go to the clients’ pool.

In order to make this happen, Ripple Swim Academy uses web and mobile software to manage clients, while providing built-in incentives designed to keep students on track and engaged.

Key Points:

  • Giphy API
  • Robust Database
  • RESTful API
  • Open cloud platform to incorporate iOS development

Puppy Connect

This marketplace web application allows users to display items for sale on an open thread. Though the wireframe suggested creating a bicycle marketplace, I tailored this web app to fit the needs of puppy owners who need to sell newborn puppies.

This application was built using the MEAN stack. Key functionalities included are:

  • Registration validation
  • Search
  • Image upload
